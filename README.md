This tool is designed to help students with **Dyslexia** or other reading needs by making digital textbooks more accessible. It "unlocks" the book reader so you can use text-to-speech tools like **Speechify**.

---

## üßê What is this?
Many digital textbooks block your ability to right-click or highlight text to prevent copying. However, this also blocks helpful tools like **Speechify** from reading the text aloud to you. 

This script is a **Userscript**‚Äîa tiny piece of helper code that runs only on your computer. It tells the website to stop blocking your mouse, allowing you to highlight words and listen to them as you study.

## üõ†Ô∏è How to Install (Step-by-Step)

To use this script, you need to install a "Manager" in your browser called **Violentmonkey**.

### Step 1: Install Violentmonkey
1. **Open your web browser** (Google Chrome, Microsoft Edge, or Firefox).
2. **Visit the store link** for your browser below:
   * [Chrome Web Store](https://chrome.google.com/webstore/detail/violentmonkey/jinjaccalgkgednnccohejagnpfdpano)
   * [Microsoft Edge Addons](https://microsoftedge.microsoft.com/addons/detail/violentmonkey/eeagobolheoikdgbbhcdaadhnoingjbn)
   * [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/)
3. Click the **Add** or **Get** button to install it.
4. You will see a new **Monkey Icon** appear in your browser's toolbar (usually top-right).



### Step 2: Add the Unlocker Script
1. Click the **Violentmonkey icon** (the monkey face) in your toolbar.
2. Click the **Plus (+)** button or select **"Create a new script"**.
3. A new screen will open with some text already in it. **Delete all the existing text**.
4. **Copy the script code** provided to you and **paste** it into that empty window.
5. Click **Save & Close** at the top of the screen.

### Step 3: Start Reading
1. Go to your book on **brighteducation.io**.
2. **Refresh the page** (click the reload button or press `F5`).
3. You can now highlight text and right-click as normal! Speechify will now be able to "see" and read the text to you.

## üí° Frequently Asked Questions

* **Is this safe?** Yes. This script only changes how the website behaves for *you* so you can read more easily. it does not share your data or passwords.
* **How do I turn it off?** Click the Monkey Icon and click the toggle switch next to "BrightEducation Reader Unlocker" to turn it off.
* **Does this work on other sites?** This specific script is custom-built for BrightEducation, but similar scripts can be used for other restricted sites.
